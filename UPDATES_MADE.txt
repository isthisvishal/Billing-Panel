QUICK REFERENCE:
================================================================================

FOR YOUR VM/SERVER:
  SSH into your Linux server, then run:
  
  curl -fsSL https://github.com/isthisvishal/Billing-Panel/raw/main/scripts/install.sh | sudo bash

  Answer 2 prompts:
    1. Enter your domain (e.g., billing.example.com)
    2. Enter MySQL password (make it strong!)

  Wait 15 minutes. Visit https://your-domain.com. Done!


MANUAL ALTERNATIVE:
  git clone https://github.com/isthisvishal/Billing-Panel.git
  cd Billing-Panel
  cp .env.example .env
  nano .env                    # Edit: APP_URL, DB_PASSWORD
  nano Caddyfile               # Edit: domain
  docker compose up -d --build
  sleep 20
  docker compose exec app php artisan migrate --force
  docker compose exec app php artisan app:install


COMMON COMMANDS:
  docker compose ps            # View all services
  docker compose logs -f app   # View application logs
  docker compose restart       # Restart everything
  docker compose down          # Stop all services
